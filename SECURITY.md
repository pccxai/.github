# Security Policy

## Reporting

Do not file public issues for security findings.

Send reports to k1h6w4@gmail.com with:

- Affected repository and commit, tag, or release.
- Reproduction steps or a minimal proof of concept.
- Impact and known workarounds, if any.

Maintainers acknowledge reports within 7 days and coordinate disclosure after triage.

## Scope

This placeholder applies to public repositories under the `pccxai` organization unless a repository provides its own security policy.
