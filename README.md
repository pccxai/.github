# PCCX AI

Open hardware/software research for NPU architecture, FPGA acceleration, and AI-assisted verification.

## Projects

- `pccx` — SystemVerilog-based NPU core architecture
- `pccx-lab` — AI-assisted development, verification, visualization, and reporting toolchain
- `pccx-FPGA-NPU-LLM-kv260` — KV260 FPGA integration with NPU + LLM/KV-cache experiments

## Current Focus

- Hardware/software co-design for edge AI
- NPU RTL architecture and verification
- FPGA deployment on Xilinx KV260
- AI-assisted design, analysis, and documentation workflows
