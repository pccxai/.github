# pccxai

pccxai is an open hardware and software organization for the pccx architecture, with current public work split across the [v002 KV260 RTL line](https://github.com/pccxai/pccx-FPGA-NPU-LLM-kv260), [pccx-lab verification tooling](https://github.com/pccxai/pccx-lab), the [v003 LLM line](https://github.com/pccxai/pccx-LLM-v003), and the [vision-v001 line](https://github.com/pccxai/pccx-vision-v001); release wording is tied to repository evidence, execution is tracked on the [pccxai project board](https://github.com/orgs/pccxai/projects/1), and the public home page lives in [pccxai/pccxai](https://github.com/pccxai/pccxai).
